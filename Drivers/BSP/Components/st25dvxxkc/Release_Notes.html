Purpose

The ST25DVXXKC Component Driver is a driver for the STM32Cube. This software implements a Component Driver running on STM32 for STM’s NFC. The ST25DVXXKC devices are NFC RFID tags offering upto 64Kbit of electrically erasable EEPROM. It is built on top of STM32Cube software technology that ease portability across different STM32 micro-controllers

The expansion is built on STM32Cube software technology to ease portability across different STM32 microcontrollers.

    Here is the list of references to user documents

  • Datasheet : Dynamic NFC/RFID tag IC with 4-Kbit, 16-Kbit or 64-Kbit EEPROM, fast transfer mode capability and optimized I2C
